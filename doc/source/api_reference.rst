pdbcraft consists of several modules:

* :doc:`structure <structure>`, containing utilities to manipulate molecular structures.

* :doc:`pdbparsers <pdbparsers>`, containing parsers for PDB files.

* :doc:`mmcifparsers <mmcifparsers>`, containing parsers for PDBx/MMCIF files.

* :doc:`pdbwriters <pdbwriters>`, containing writers for PDB files.

* :doc:`mmcifwriters <mmcifwriters>`, containing writers for PDBx/MMCIF files.

* :doc:`fastawriters <fastawriters>`, containing writers for FASTA files.

* :doc:`util <util>`, containing miscellanea utility functions.
