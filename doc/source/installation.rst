Installing pdbcraft
===================

.. toctree::
   :maxdepth: 1
   :hidden:

   installation_venv
   installation_conda


pdbcraft is a Python package requiring **Python 3.11 or higher**.

Please ensure that you have Python installed before proceeding with the installation.

The package has been tested only on Unix-based systems.

We provide installation instructions that assumes Python 3.11 is installed, but the same steps remain valid for later versions.

Here, we provide instructions for installing pdbcraft in:

*  A :doc:`virtual environment using virtualenv <installation_venv>`.
*  A :doc:`virtual environment using conda <installation_conda>`.
